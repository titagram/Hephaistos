# Autopoiesis Skill — Global Telos & Meta-Observation

This skill allows Hades to interact with the global Autopoiesis Observer, inspect suggestions, record structured notes, and conduct Telos Workshops.

## Commands

- `/autopoiesis`: Start or resume the Telos Workshop to define or refine Hades's global purpose, priorities, and prohibitions.
- `/autopoiesis suggestions`: List current global suggestions generated by the Opportunity Observer.
- `/autopoiesis show <suggestion-id>`: Display detailed evidence, score, and explanation for a suggestion.
- `/autopoiesis note`: Record a user-reviewed observation note into the global ledger.
- `/autopoiesis telos`: View the current active Telos revision and status.

## Governance & Approval Rules

1. **Telos Drafts Are Inert**: Hades can draft Telos amendments or conduct workshops, but **cannot** activate them independently.
2. **Interactive Host Approval**: Activation strictly requires the user to execute:
   `hermes evolution telos approve <digest>`
3. **No Unsafe Operations**: Suggestions or Telos revisions cannot grant network access, leak secrets, or mutate core files without user consent.
