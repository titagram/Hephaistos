# Technical Indexes

Questi indici sono mappe operative iniziali, non cataloghi esaustivi di tutto
il repository. Quando una superficie e dinamica o troppo ampia, il documento
indica il comando `rg` da usare per rigenerare l'elenco.

Indici:

- `ROUTES_OR_ENTRYPOINTS.md` - CLI, slash command, FastAPI, React routes,
  gateway/TUI/desktop entrypoint.
- `DATA_MODEL.md` - tabelle, file state e persistenza osservati.
- `SIDE_EFFECTS.md` - subprocess, filesystem, rete, cron, delivery, webhook.
- `DEPENDENCIES.md` - package manager, lockfile, runtime e script.
- `SECURITY.md` - confini, credenziali, auth, rischi e guardrail.
